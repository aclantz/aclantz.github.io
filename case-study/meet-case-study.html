A React application giving users information about events in certain cities. Allows users to search for cities and displays visual data about what events are available and specifics about each event.

I used a full Test Driven Development process to build this application. Learning to create functions in this format really allows you to see what testing can do for a complicated application, and how it makes it much easier to maintain clean code.

The back end of this application was built using serverless functions (AWS). I learned the format for implementing and deploying serverless functions and the testing needed to implement them in a TDD application.

I am very proud of the layout and look of this application. I think the choice to have the "Show Details/Hide Details" button on the bottom and a part of the border of the Event helped it to look more professional and clean.
